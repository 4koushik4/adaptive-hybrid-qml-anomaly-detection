# QML Real-Time Anomaly Detection System

A Real-Time Quantum Machine Learning (QML) framework for anomaly detection in:
- Cybersecurity
- IoT systems
- Industrial monitoring

The system combines:
- Classical preprocessing
- Variational Quantum Circuits (VQC)
- Real-time streaming
- Drift detection
- Noise robustness analysis
- Live dashboard visualization

---

# Features

- Real-time anomaly detection
- Quantum-enhanced pattern recognition
- Variational Quantum Circuit (VQC)
- Hybrid Quantum-Classical Models
- QSVM & QNN implementations
- Multi-source streaming simulation
- Drift detection
- Noise injection testing
- Live Streamlit dashboard
- Benchmarking against classical ML models

---

# Supported Datasets

- NSL-KDD
- UNSW-NB15
- MQTT-IoT
- Simulated streaming data

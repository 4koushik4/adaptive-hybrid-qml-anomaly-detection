# Methodology

## Introduction

This project follows a hybrid quantum-classical machine learning methodology for detecting anomalies in real-time streaming environments.

The methodology integrates classical preprocessing pipelines with Variational Quantum Circuits (VQC) to improve anomaly detection performance on high-dimensional and noisy datasets.
